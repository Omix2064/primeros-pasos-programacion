# Mi Primer Proyecto de Ingeniería
Este es un pequeño programa en Python para comparar dos valores.

## ¿Qué hace?
- Pide dos números (A y B).
- Calcula cuál es el mayor.
- Obtiene la diferencia absoluta entre ambos.

## ¿Para qué sirve esto en el mundo real?
Es la base para calcular desviaciones en sensores de temperatura o presión.
